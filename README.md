## Some Powershell or cmd notes for running the simulation:
- Running sims in powershell:
```
cmd.exe /c wCaseName_win64_GPU.bat
```
However, using command prompt is simpler.
- Clear case bat file:
```
rmdir "./CaseFolderName" /S /Q
```
